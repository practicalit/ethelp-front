Hi,

Wonder what this is all about?

Checkout this page to learn more.

In short, this is a platform to help out and to get help to and from fellow Ethiopians

Do you, or someone you know need help?

Go to help page and do post the needs. It is fast!

Are you IT Professional?

If you are developer, front-end guru, DBA, dev-ops engineer, there is a lot you can help us.

This is an ongoing project, you can pick it from anywhere and work on it.

  • Install postres locally or use docker for postgres. Docker is preferable
  • Play with the app locally
  • Add tests, features, corrections, bug fixes..
  • Do a pull request on github and another fellow member will review and merge it.
  • Let's do it.

What is included as feature right now - MVP

As of now, the list is limited to MVP. Just what is needed to launch it. There are others stuffs to be added of course. But sticking with the basics, these are the initial implementations.

  • Registration for the new member
  • Token Based Login. This is an industry standard oAuth extensible implementation
  • Create a new help post. This can be for self or others that one know need a help
  • Adding supporting images for the help
  • Allowing discussions for the help(comment)
  • Depositing money. Once deposited, the member can use it to help others as s/he wishes
  • Donating to the help provided s/he has the deposit already

Technologies

  • Programming Language: Java
  • gradle for package management
  • Framework used: Spring[Boot]
  • File storage: AWS S3
  • Database: Postgres
  • Cache: Redis
  • Frontend - Bootstrap, jQuery, node/npm
  • git/hub for code management
